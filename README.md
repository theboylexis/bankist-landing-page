# Bankist Landing Page

A modern, clean, and interactive landing page for a minimalist banking service. Built with HTML, CSS, and vanilla JavaScript, this project showcases best practices in UI/UX design and frontend development.

## Features

- Sticky navigation for seamless site navigation  
- Smooth scrolling and animated section reveals for polished UX  
- Tabbed components to display banking operations clearly  
- Lazy-loading images for improved performance  
- Responsive modal window for quick user sign-up  
- Testimonial slider with navigation controls  
- Uses CSS variables for easy theme customization  
- Implements Intersection Observer API for sticky nav and reveal-on-scroll effects
